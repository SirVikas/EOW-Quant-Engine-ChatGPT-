# 🥉 PART 3 — SCALING ENGINE (CRITICAL)

## 📁 scaling_engine.py

### Logic:

```text id="x3m7q1"
If strategy performs well → increase capital
If underperforms → reduce exposure
```

---

### Rules:

```text id="b8k4p6"
Scale UP:
- PF rising
- DD low
- Stability high

Scale DOWN:
- DD rising
- PF dropping
```

---

### Example:

```python id="y4n8c2"
def scale(position_size, performance):
    if performance.good:
        return position_size * 1.2
    return position_size * 0.8
```
