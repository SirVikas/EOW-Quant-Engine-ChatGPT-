# 🔒 GLOBAL RULES (NON-NEGOTIABLE)

```text id="r3n8p4"
1. Max step change per cycle ≤ 10% of parameter range
2. One change at a time (no multi-variate jumps initially)
3. Mandatory sandbox validation before live
4. Mandatory monitoring window (≥ 10–20 trades)
5. Automatic rollback on guardrail breach
6. Full versioning of every change
7. Human kill-switch always available
```

# 🥉 PART 3 — CHANGE PLANNER (STEPWISE)

## 📁 change_planner.py

```python id="d2v7s8"
def plan_change(param, current, target):
    step = (target - current) * 0.25   # 25% of delta
    return current + step
```

❗ First iteration = **partial move**, not full

# 🛡 PART 4 — GUARDRAIL ENGINE

## 📁 guardrail_engine.py

### Hard Stops:

```text id="g1x6q3"
- Max DD increase > +3%
- PF drop below threshold
- Risk-of-Ruin increase > limit
- Consecutive losses spike
```

```python id="h7k3n9"
def safe_to_proceed(metrics):
    return metrics.dd_delta < 0.03 and metrics.pf > min_pf
```
