## 1. 🔴 CONTRADICTION DETECTOR

### Function:

```python
detect_contradictions(report_data) -> list[dict]
```

### MUST DETECT:

```text
✔ signals > 0 AND trades = 0
✔ capital_idle > 70% AND trades = 0
✔ rejection reasons exist BUT root cause = none
✔ PF < 1 BUT system marked "normal"
✔ session_trades = 0 BUT win_rate populated
```

---

## 2. 🧠 ROOT CAUSE RESOLVER (REWRITE)

### Function:

```python
resolve_root_cause(data, contradictions) -> dict
```

### LOGIC PRIORITY:

```text
1. GATING BLOCK (SLEEP_MODE, LOW_SCORE, RR_FAIL)
2. SIGNAL QUALITY (low score distribution)
3. RISK BLOCK (halt / DD)
4. CAPITAL UNDERUTILIZATION
5. TRUE NO-OPPORTUNITY
```

### EXAMPLE OUTPUT:

```text
PRIMARY:
"Trade execution blocked by SLEEP_MODE despite active signal flow"

SECONDARY:
"Score threshold not met for majority of signals"
```

---

## 3. 🔁 SIGNAL FLOW VALIDATOR

### Function:

```python
validate_signal_flow(data) -> dict
```

### MUST OUTPUT:

```text
✔ % signals rejected
✔ % signals passed
✔ execution gap (signals → trades)
✔ dominant rejection reason
```

---

## 4. ⚖️ SESSION vs HISTORICAL SPLITTER

### Function:

```python
split_metrics(data) -> dict
```

### REQUIRED:

```text
Session:
- trades
- PnL
- win rate

Historical:
- total PnL
- lifetime stats
```

❗ MUST NEVER MIX
