# 🔴 PART 2 — NET EDGE FILTER (CRITICAL DECISION LAYER)

## 📁 File

core/alpha/net_edge_engine.py

---

## 🔧 Logic

```python
expected_move = signal.expected_move_pct

cost = cost_engine.estimate_cost(...)

net_edge = expected_move - cost["cost_pct"]

if net_edge < cfg.MIN_NET_EDGE:
    reject(reason="NEGATIVE_NET_EDGE")
```

---

## 📌 Config

```text id="cfg"
MIN_NET_EDGE = 0.10% (initial)
EXPLORATION_ALLOW_NEGATIVE = True (small size only)
```
